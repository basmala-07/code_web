OOP: Classes & Objects

C# OOP relies on classes, properties, and methods to model real-world entities cleanly.

Source Code

class Car
{
    public string Brand { get; set; }
    public int Year { get; set; }

    public void ShowInfo()
    {
        Console.WriteLine($"{Brand} - {Year}");
    }
}

Car myCar = new Car { Brand = "Toyota", Year = 2024 };
myCar.ShowInfo();

Console Result

Toyota - 2024

OOP: Inheritance

Source Code

class Animal
{
    public void Eat()
    {
        Console.WriteLine("Animal is eating");
    }
}

class Dog : Animal
{
    public void Bark()
    {
        Console.WriteLine("Dog is barking");
    }
}

Dog d = new Dog();
d.Eat();
d.Bark();

Console Result

Animal is eating
Dog is barking

OOP: Encapsulation (get; set;)

Source Code

class Student
{
    public string Name { get; set; }
    public int Level { get; set; }
}

Student s = new Student();
s.Name = "Lina";
s.Level = 2;
Console.WriteLine($"{s.Name} - Level {s.Level}");

Console Result

Lina - Level 2
